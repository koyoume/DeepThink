# DeepThink 설계 프로토타입

라이브 웹앱의 시각·구조를 반영한 설계 프로토타입 모음(설계문서). 각 변경 논의 시 여기에 버전을 쌓는다.

## 규칙
- **실제 산출물 밀착**: 이상적 목업이 아니라 실제 앱의 구조·토큰을 그대로 반영(배경 `#F1ECE2`, 브랜드 보라 `#6B4EFF`, 세리프 제목 Fraunces+Noto Serif KR, FAB, check/comment 글리프, 순서기반 카테고리 색, 제목 자동축소 등). 구조·동작은 `../legacy-prototype/UI-DESIGN.md §4~6` 고정.
- **파일명에 버전**: `deepthink-proto-vNN-<요약>.html` — 생성 순서 추적.
- **생성 즉시 push**: 만들면 바로 이 폴더에 커밋·푸시(main, 브랜치 없음).
